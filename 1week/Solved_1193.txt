등차수열의 합 공식을 사용하여 푸는 문제이다.
해당 문제의 표를 보면 라인이 홀수일때는 분자가 감소하고 분모가 증가하며
라인이 짝수이면 분자가 증가하고 분모가 감소한다.
이 때, 분자 + 분모 = 라인 + 1을 참고해서 문제를 풀면 된다.


1)찾고 싶은 분수의 위치를 X로 입력 받는다.
2)반복문을 통해서 cnt의 값이 X보다 작다면 계속해서 line의 값을 증가시키며 
  cnt에 등차수열 공식을 이용한 line * (line + 1) / 2의 값을 대입한다.
3)lineCnt(라인에서 X가 원하는 분수 위치)에 line - 1개의 분수들의 개수를 X에 뺀 값을 대입한다.
4)만약 line이 짝수이면 x에는 lineCnt의 값을 y에는 line + 1 - x의 값을 준다.
  이 때 y에 line + 1 - x를 하는 이유가 "분자 + 분모 = 라인 + 1"이기 때문이다.
  짝수가 아닌 홀수이면 y에 lineCnt의 값을 x에는 line + 1 - y의 값을 준다.
